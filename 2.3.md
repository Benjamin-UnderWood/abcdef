# 3. Postman 使用详解

**命令行下**

* wget
* curl
* httpie
* coreapi





**应用程序**

* postman





* 参数
* header
* 认证机制
* 响应内容,格式化高亮



**environment**

* 里面写一些模板, 然后url 里面通过$1 替换



**请求**

* params
* anthorization 授权
* headers http报文头部
  * Temporary Headers 默认8个请求头
* body 主体
  * form-data 模拟表单
  * x-www-form-urlencoded 发送文件
* raw 自己编写 在后面下拉框可选择格式
* binary 二进制文件
* Pre-request-Script 预请求 发送之前的脚本
  * 右侧有代码片段 例如设置全局变量
* Tests
  * 先不发送, 先测试

* Cookies  code
* 响应状态码
